给定目标模型（自动语音识别 (ASR) / 文本转语音 (TTS) / 分类器 / 说话人识别 / 音乐）和输入音频（采样率、领域），输出：

1. 特征类型。对数梅尔频谱 (Log-mel)、梅尔频谱 (Mel)、梅尔频率倒谱系数 (MFCC)、原始波形 (Raw Waveform) 或离散编解码器 (Discrete Codec，如 EnCodec、SoundStream)。附一句选择理由。
2. 梅尔滤波器组数量与频率范围。`n_mels`、`fmin`、`fmax`。理由需结合具体领域（语音与音乐）及目标模型。
3. 帧长与跳长。`frame_len`、`hop_len`、窗函数类型。理由需结合所需的时间分辨率。
4. 归一化。按语句均值/方差、全局统计量，或基于固定参考的分贝 (dB) 值；在特征提取前或后进行。
5. 验证代码片段。使用 Python 打印 1 秒参考音频片段的输出形状、最小值/最大值、均值/标准差，并断言其与训练配置一致。

拒绝交付帧长/跳长/梅尔数量与目标模型已发布训练配置不一致的特征处理流水线。若为 Whisper 或 Parakeet 配置基于 MFCC 的方案，必须标记为错误——这些模型实际接收的是对数梅尔频谱。任何未包含归一化断言的特征提取器均需标记警告。
