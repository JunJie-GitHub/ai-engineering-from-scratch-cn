# 不平衡数据策略

这是一份用于处理不平衡分类问题的决策清单。请按照以下步骤，为你的问题选择最合适的方法。

## 步骤 1：评估不平衡程度

- 统计每个类别的样本数量
- 计算不平衡比例（多数类 / 少数类）
- 轻度：比例 < 3:1（例如 70/30）
- 中度：比例 3:1 至 20:1（例如 95/5）
- 重度：比例 > 20:1（例如 99/1）

## 步骤 2：选择合适的评估指标

对于不平衡数据集，应优先使用精确率（Precision）/召回率（Recall）/F1分数（F1 Score），而非准确率（Accuracy）。请根据具体问题选择：

| 场景 | 主要指标 | 次要指标 |
|-----------|---------------|-----------------|
| 漏报正样本代价极高（如欺诈检测、疾病诊断） | 召回率 | F2分数（F2 Score） |
| 误报代价极高（如垃圾邮件过滤、推荐系统） | 精确率 | F0.5分数（F0.5 Score） |
| 两者重要性大致相当 | F1分数 | 马修斯相关系数（Matthews Correlation Coefficient, MCC） |
| 需要单一排序指标 | 精确率-召回率曲线下面积（Area Under Precision-Recall Curve, AUPRC） | ROC曲线下面积（Area Under ROC Curve, AUC-ROC） |
| 需要跨数据集比较 | 马修斯相关系数 | 精确率-召回率曲线下面积 |

## 步骤 3：选择重平衡策略

### 按不平衡程度划分

| 不平衡程度 | 首选尝试 | 备选尝试 | 避免使用 |
|-----------|-----------|------------|-------|
| 轻度（< 3:1） | 类别权重（Class Weights） | 阈值调优（Threshold Tuning） | 过采样（Oversampling）（不必要） |
| 中度（3:1 至 20:1） | SMOTE + 类别权重 | 叠加阈值调优 | 欠采样（Undersampling）（数据损失过大） |
| 重度（> 20:1） | SMOTE + 类别权重 + 阈值调优 | 结合平衡装袋法（Balanced Bagging）的集成学习 | 仅使用欠采样 |

### 按数据集规模划分

| 数据集规模 | 推荐策略 | 原因 |
|-------------|-------------------|--------|
| < 1,000 个样本 | 过采样或 SMOTE（合成少数类过采样技术） | 无法承受多数类数据丢失 |
| 1,000 - 10,000 个样本 | SMOTE + 阈值调优 | 少数类样本足以支持 k近邻算法（k-NN） |
| > 10,000 个样本 | 类别权重或欠采样 | 速度快，且少数类数据充足 |

## 步骤 4：应用具体技术

### 类别权重（建议优先尝试）
- 在 scikit-learn 中：`class_weight='balanced'`
- 无需修改原始数据
- 适用于任何基于损失函数（Loss Function）的模型
- 在数学期望上等价于过采样

### SMOTE
- 仅应用于训练集（切勿用于测试集/验证集）
- 使用 k=5 个近邻（默认值）
- 与类别权重结合使用效果最佳
- 注意决策边界附近可能产生的噪声合成样本

### 阈值调优
- 训练模型，并在验证集上获取预测概率
- 在 0.05 至 0.95 范围内遍历阈值
- 选择能使你选定指标最大化的阈值
- 务必在验证集上调优，绝不可使用测试集

## 步骤 5：正确进行验证

- 使用分层交叉验证（Stratified Cross-Validation）（确保每个折中保留原始类别比例）
- 在原始（未重采样）测试集上报告指标
- 切勿在划分数据集前应用 SMOTE，仅应在训练折上使用
- 与“始终预测多数类”的基线模型进行对比

## 步骤 6：避免常见错误

- 在划分训练集/测试集前对整个数据集应用 SMOTE（导致数据泄露 Data Leakage）
- 使用准确率作为评估指标
- 未优先尝试类别权重（最简单的方法，通常已足够）
- 先过采样再进行交叉验证（合成样本会跨折泄露）
- 忽略阈值调优（无需重新训练即可免费提升性能）
- 在小数据集上使用随机欠采样（会丢弃过多数据）

## 快速决策树

1. 不平衡比例是否 < 3:1？ -> 仅尝试类别权重
2. 数据集是否 > 10,000 个样本？ -> 类别权重 + 阈值调优
3. 数据集是否 < 1,000 个样本？ -> SMOTE + 类别权重
4. 其他情况 -> SMOTE + 类别权重 + 阈值调优
5. 效果仍不理想？ -> 使用平衡装袋法集成模型
